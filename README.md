## Run Locally

Clone the project

```bash
  git clone https://github.com/TigranBalayan10/bm-limo
```

Go to the project directory

```bash
  cd bm-limo
```

Install dependencies from the root folder

```bash
  npm install
```

Start the server

```bash
  npm run develop
```
